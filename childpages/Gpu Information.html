Information about GPUs

Welcome to the gpu information page. Here is where you will find basic information about GPUs. And what might they be used for.

What is a GPU?

A GPU(Graphical Processing unit). Is a dedicated card designed to manipulate and alter memory to accelerate creation of images in frame buffers. The GPU is intended for output towards a display device. GPUs are also used in embedded systems, mobile phones, personal computers, workstations, and game consoles. (Additional information. A GPU does NOT use cores or threads. They use cuda cores.)
